LARAVEL APP REUSE INSTRUCTIONS

1. Extract zip file to your server/public_html directory

2. Configure .env file:
   - Copy .env.example to .env
   - Set database credentials
   - Set APP_URL

3. Install dependencies:
   composer install
   npm install

4. Generate key:
   php artisan key:generate

5. Run migrations:
   php artisan migrate

6. Seed database:
   php artisan db:seed

7. Compile assets:
   npm run build

8. Set directory permissions:
   chmod -R 775 storage bootstrap/cache

9. Configure web server to point to public folder

10. Run queue worker (if using):
    php artisan queue:work
